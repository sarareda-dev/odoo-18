This module enhances the default product search in Odoo by adding support for partial search and unordered multi-word search. It works seamlessly across:

  • Point of Sale (POS)
  • Sales Order Line
  • Purchase Order Line

For example, typing "mac m3 air" will correctly return "MacBook Air M3" products, even if the words are out of order.

Key Features

  • Partial search by any part of the product name.
  • Unordered multi-word search.
  • Works in both English and Arabic.
  • Compatible with Odoo 18.
